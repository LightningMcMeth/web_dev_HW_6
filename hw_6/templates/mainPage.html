View specified entity endpoint

example: http://127.0.0.1:8000/hw_6/entities/abc123/
This endpoint displays the specified product entity. All valid entity ids: abc123, def456, ghi789, jkl012


Delete specified entity endpoint

example: http://127.0.0.1:8000/hw_6/deleteEntity/abc123
This endpoint deletes the specified product entity. All valid entity ids: abc123, def456, ghi789, jkl012


Create entity endpoint

http://127.0.0.1:8000/hw_6/createEntity/
This endpoint allows the creation of a new product through postman

Example values:

'name': 'RTX2060', 'price': 9.99, 'category': 'PC component', 'description': 'it works', 'availability': True, 'id': 'gamer123'
